# AGENTS.md

This file provides guidance to agents when working with code in this repository.

## Project Overview
- This is a research project for Information Systems (IS) discipline
- Focus: Academic writing, systematic literature reviews, and empirical research with Scopus-indexed sources
- Main artifact: `researcher.agent.md` - defines a rigorous IS researcher agent

## Key Files & Folders
- `researcher.agent.md`: IS researcher agent configuration and guidelines
- `.concept/`: Empty folder (likely for concept maps or diagrams)
- `.vscode/`: VSCode configuration (including MCP server setup for devmemory extension)

## Document Management Workflow
- Research projects follow a specific folder structure (draft/, diagrams/, data/) as defined in researcher.agent.md
- Content is written in English (IELTS Band 8 standard) with active voice dominance
- Citations must be from Scopus/ISI-indexed sources with valid URLs

## Prohibited Behaviors
- Do not fabricate data, citations, or theoretical frameworks
- Reject blogs, Wikipedia, or non-peer-reviewed sources
- Avoid passive voice as default construction
- Do not generate generic, soulless academic prose
